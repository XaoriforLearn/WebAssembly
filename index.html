What is this?

What is this?

WebAssembly - это не ЯП, а формат представления данный в двоичной системе исчисления, предназначеный для выполнения кода в браузере, который компилируется из языков высокого уровня.

history of creation

history of creation

Всё началось примерно в 2015 году, когда представители Mozilla, Google, Microsoft и Apple, собрались вместе, чтобы обсудить будущее веб-технологий. Они понимали, что JavaScript, хотя и мощный, не всегда может обеспечить ту производительность, которая нужна для сложных приложений. Именно тогда родилась идея созданияWebAssembly

How it works

How it works

  • WebAssembly выполняется в виртуальной машине внутри браузера (Т.е не компилируется и не имеет интерпритатора).
  • WebAssembly имеет статическую типизацию и компактный бинарный формат, который может быть декодирован гораздо быстрее, чем JavaScript может быть распарсен.

Example code

Example code

Rust


							fn factorial(n: u32) -> u32 {
								if n == 0 {
									return 1;
								}
								n * factorial(n - 1)
							}

Example code

WebAssembly


						(module
						(func $factorial (param $0 i32) (result i32)
						  (if (result i32)
							(i32.eq
							  (get_local $0)
							  (i32.const 0))
							(then
							  (i32.const 1))
							(else
							  (i32.mul
								(get_local $0)
								(call $factorial
								  (i32.sub
									(get_local $0)
									(i32.const 1)))))))
						(export "factorial" (func $factorial)))
